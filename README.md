# Try esbuild and Typescript

A sample repository to try out esbuild.

## Building and publishing

    npm i
    npm run test
    npm run build
    npm login
    npm publish

## Installation

    npm i @adambirse123/try-esbuild-typescript

## Usage

```
const { sum } = require("@adambirse123/try-esbuild-typescript");

console.log(sum(1, 2));

```
